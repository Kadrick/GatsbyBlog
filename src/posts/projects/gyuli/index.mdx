### 개요

#### 기술 스택

- Java Spring Boot
- MySQL

#### 프로젝트 소개

- 개발 기간: 2024.10 ~ 2024.12
- 외주 진행 프로젝트 (Back-end 개발)
- 감귤 재배 농가에서 기후 데이터와 품종별 재배 환경을 데이터베이스화하여 효율적인 영농 기록 시스템 개발
- 농가에서 증빙 자료로 즉시 활용할 수 있도록 표준 양식의 PDF 리포트 변환 기능 개발

---

### 주요 작업

- REST API 및 DB 스키마 구조 설계
- 관리자 대시보드 구성 작업
- 등록된 농가들의 작성 현황, 기간별/품종별 데이터 조회를 위한 시스템 개발
- 오픈 소스를 이용해 일지 정보를 HTML 템플릿으로 저장하고 이를 PDF로 변환하는 모듈 개발
