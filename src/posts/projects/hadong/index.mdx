### 개요

#### 기술 스택

- Java Spring Boot
- MySQL

#### 프로젝트 소개

- 개발 기간: 2024.08 ~ 2024.10
- 외주 진행 프로젝트 (Back-end 개발)
- 공장 근무자의 안전을 위해 안전보건 자료(요약서, MSDS 전문/요약본)과 안전보호구 착용 기준, 안전 제보 기능, 공지사항 기능 개발
- 비콘을 활용해 위험구역 접근 시 경고음을 발생시키는 시스템 개발

---

### 주요 작업

- REST API 및 DB 스키마 구조 설계
- 관리자 대시보드 구성 작업
- 현장별/구역별 알림 수신 정책(Push/Sound)을 동적으로 제어할 수 있는 관리자 설정 기능 개발
- 비콘 신호 수신 현황, 위험 구역 진입 로그를 파악할 수 있는 기능개발
- 화학물질 별 상세 조회를 위한 QR코드 생성 모듈 개발
