## 개요

### 기술 스택

- TypeScript, React (VS Code Webview)
- Python, FastAPI
- PostgreSQL
- RabbitMQ
- VSIX Build Tools
- GitHub Actions

### 프로젝트 소개

- **개발 기간:** 2025.03 ~ 2025.07
- **역할:** 팀장 (캡스톤 프로젝트)
- **목표:** 개발 과정에서 가장 병목인 코드·구조 분석 단계를 자동화하는 VS Code Extension 개발

  }
  result={

    </>
  }

<CaseStudy
  id="case-2-vsix-fastapi"
  title="Case 2. VSIX 빌드 툴을 활용한 FastAPI 백엔드 통합 및 마켓플레이스 배포"
  problem={
    <>
      VS Code Webview 환경과 Python 기반 AI 백엔드 서버 간 연동 구조가 복잡했다.
      Extension 패키징·배포 파이프라인이 없어 프로덕트 릴리즈까지의 경로가 불명확했다.
    </>
  }
  action={
    <>
      VSIX Build Tool로 Extension 패키징을 최적화하고, FastAPI 백엔드를 CI/CD(GitHub Actions)로 연동했다.
      VS Code Marketplace 공식 배포 절차를 완료했다.
    </>
  }
  application={
    <>
      Webview ↔ Extension Host ↔ FastAPI REST API 3-tier 통신 구조를 구성했다.
      분석 요청 병목 방지를 위해 RabbitMQ 기반 비동기 처리도 함께 도입했다.
    </>
  }
  result={
    <>
      글로벌 VS Code Marketplace에 실제 프로덕트를 릴리즈하는 경험을 확보했다.
    </>
  }
/>
