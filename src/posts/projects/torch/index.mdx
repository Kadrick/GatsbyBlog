## 작업 내용

### 팀 운영 · 일정 관리

- **마일스톤 설계** — Extension, Backend, AI 연동 파트별 역량에 맞춰 통합 일정을 수립
- **주간 정기 회의** — 파트 간 기술적 블로커를 즉시 해소하고 진행 상황을 공유
- **인터페이스 사전 합의** — API 스펙·Webview 메시지 프로토콜을 초기에 고정해 병렬 개발이 가능하도록 조율
- **산학협력 경진대회** — 정시 배포를 완료하고 우수상 수상

### VS Code Extension · Webview

- **Extension Host 개발** — 워크스페이스 코드 컨텍스트 수집 및 분석 요청 흐름 구현
- **React Webview UI** — 분석 요청·결과 표시를 위한 Webview 화면 구성
- **3-tier 통신 구조** — Webview ↔ Extension Host ↔ FastAPI REST API 연동

### 백엔드 · AI 연동

- **FastAPI 서버 구축** — 분석 요청 수신·응답 API 및 분석 이력 저장
- **Gemini API 연동** — 코드 컨텍스트를 전달하고 문서화 결과를 반환하는 AI 파이프라인 구성
- **RabbitMQ 비동기 처리** — 분석 요청 병목을 줄이기 위해 큐 기반 Worker 처리 도입
- **PostgreSQL** — 분석 요청·결과 이력 관리

### 배포 · 릴리즈

- **VSIX 패키징** — VSIX Build Tool로 Extension 빌드·패키징 자동화
- **CI/CD** — GitHub Actions로 백엔드 배포 파이프라인 구성
- **Marketplace 배포** — VS Code Marketplace 공식 배포 절차 완료
