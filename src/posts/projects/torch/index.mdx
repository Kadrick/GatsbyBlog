### 개요

#### 기술 스택

- TypeScript, React
- Python, FastAPI
- PostgreSQL
- RabbitMQ
- VSIX Build Tools

#### 프로젝트 소개

- 개발 기간: 2025.03 ~ 2025.07
- 학교 내 캡스톤 강의에서 팀장으로 진행한 프로젝트
- 개발 과정에서 가장 병목 현상이 심한 코드 및 구조 분석 단계를 효율화하기 위해, Gemini를 활용하여 프로젝트 자동 분석 및 문서화를 진행해주는 VSCode Extension 개발

---

### 주요 작업

- 분석 요청의 병목 현상을 방지하기 위해 MQ를 도입함
- 배포 서버 설정 및 Backend 서버 CI 설정 진행 (Github Actions)
- VSIX Build Tools 를 이용해 프론트엔드 결과물을 Visual Studio Code 확장으로 빌드 및 배포
