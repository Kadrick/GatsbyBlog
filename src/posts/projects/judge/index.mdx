### 개요

#### 기술 스택

- TypeScript, React
- Go
- PostgreSQL
- RabbitMQ
- Github Actions
- Docker

#### 프로젝트 소개

- 개발 기간: 2024.08 ~ 2026.08

- 학부 연구생 참여 후 시작한 프로젝트 (1인 개발)
- 교내 강의에서 사용하기 위한 온라인 채점 시스템 개발
- 코드 채점 외에 문제 제출 및 성적 통계 기능 제공

---

### 주요 작업

- 비동기 처리 기반 성능 최적화를 진행
  - 과제 제출 마감 시점의 트래픽 폭주에 대응하기 위해 메세지 큐 기반의 비동기 채점 아키텍처를 설계하여 서버 부하 분산 및 응답성 개선
- 데이터 모델링 및 UI/UX 고도화
  - 사용자 경험 개선을 위한 UI 리뉴얼 및 채점 데이터 관리 효율화를 위한 DB 스키마 재설계
  - 교수 및 조교용 대시보드를 개발하여 실시간 채점 현황 모니터링, 성적 통계 시각화 및 과제 관리 기능 제공
- 시스템 확장성 확보
  - Java, Python 등 다양한 채점 지원을 위한 엔진 확장 및 런타임 환경 구축
- 실제 사용자 피드백을 기반으로 한 성능 모니터링 및 신규 기능 업데이트 진행 중
